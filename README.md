# Quant Investment Simulator

This project simulates a rule-based quantitative investment strategy to evaluate
how systematic decisions perform over time compared to emotional investing.

## Objective
Build a simple but extensible investment simulator that:
- Applies objective trading rules
- Manages capital automatically
- Can be tested with simulated or real data

## Strategy
- Moving average crossover
- Fully invested or fully in cash
- No leverage
- Long-only

## Why this project?
This project demonstrates:
- Problem modeling
- Basic quantitative finance concepts
- Clean, executable Python code
- Ability to explain results clearly
